## Descrição

O site é um portal turistico simples desenvolvido como projeto avaliativo da disciplina de Desenvolvimento Web usando HTML e CSS. O site apresenta destinos turisticos brasileiros, gastronomia regional, informações culturais do país e algumas dicas de viagem e segurança.
